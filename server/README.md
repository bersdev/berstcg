# Backend

Used:
* express;
* typescript;
* bcrypt;
* dotenv;
* postgres;
* docker compose;

## Install dependencies

Node 21
```shell
curl -fsSL https://deb.nodesource.com/setup_21.x | sudo -E bash - &&\
sudo apt-get install -y nodejs
```

npm
```shell
sudo apt install npm
```

Node dependencies
```
npm install
```

## Start

Start server
```shell
npm run start
```

Start dev server
```shell
npm run dev
```

Build
```shell
npm run build
```
